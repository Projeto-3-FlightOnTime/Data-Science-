# Notebook 01 — EDA Flight On-Time

## Descrição
Este notebook realiza a Análise Exploratória de Dados (EDA) do projeto **Flight On-Time**, com foco em entender a distribuição das variáveis, qualidade dos dados e padrões relevantes relacionados a atrasos de voos.

## Principais atividades
- Carga e organização dos dados
- Tratamento de variáveis categóricas e numéricas
- Análises estatísticas descritivas
- Visualizações para identificação de padrões e outliers

## Bibliotecas utilizadas
- Python 3.11.2
- pandas 2.2.3
- numpy 1.24.0
- matplotlib 3.7.5
- seaborn 0.11.2
- gdown n/a
